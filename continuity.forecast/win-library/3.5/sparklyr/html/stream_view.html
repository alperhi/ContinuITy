stream_view {sparklyr}R Documentation

View Stream

Description

Opens a Shiny gadget to visualize the given stream.

Usage

stream_view(stream, ...)

Arguments

stream

The stream to visualize.

...

Additional optional arguments.

#' @examples

library(sparklyr)

dir.create("iris-in") write.csv(iris, "iris-in/iris.csv", row.names = FALSE)

stream_read_csv("iris-in/") stream_write_csv("iris-out/") stream_view() stream_stop()
